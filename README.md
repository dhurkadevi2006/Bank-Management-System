# Bank-Management-System
Java Console Based Bank Management System

## Project Description

This is a **Java console-based Bank Management System** that simulates basic banking operations like account creation, deposit, withdrawal, and transaction tracking.

## Features

* Admin Login System
* Create & Delete Accounts
* Deposit & Withdraw Money
* Transfer Money Between Accounts
* Transaction History Tracking
* Unique Account Number Generation

##  Technologies Used

* Java (OOP Concepts)
* ArrayList
* UUID for Transaction IDs

##  How to Run

1. Compile all files:

   ```
   javac *.java
   ```
2. Run the main file:

   ```
   java main
   ```
